# Nexora-SaaS (Static prototype)
This repo contains a small static prototype for *Nexora* — a next-gen SaaS landing site and simple pages.

Files:
- index.html — main landing page (multilingual toggle)
- contact.html — contact form placeholder
- dashboard.html — demo dashboard
- login.html — demo login
- business.css — core styles
- robots.txt

Usage:
- Keep these files in the repository root.
- GitHub Pages: Settings → Pages → Deploy from `main` branch, root (already configured).
- For a real site: replace placeholder forms with a backend or form service, add analytics, add structured data (JSON-LD), and replace demo assets.
